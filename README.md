# train_window_trip
### ■サービス概要

いつでもどこでも電車で旅行をする体験ができるアプリです。

### ■ユーザーが抱える課題

- 電車から見える景色をゆっくり見たいが車内が混雑していて座席に座ってゆっくり見れない。
- 秘境地域を走る路線の車窓は直接見に行きずらい。
- 友達や誰かと一緒に旅行してる時にゆっくり景色を見れない。

### ■課題に対する仮説

- 地域（都道府県）・路線名を検索条件にしてそれにヒットする車窓情報を一覧で表示することで、見たい景色の情報に簡単に辿りつけるのでは？
- 部屋の中で車窓動画を見ることで疑似体験ができるのではないか？

### ■解決方法

- 地域・路線名の条件を選択して絞り込み検索ができるようにする。
- youtubeから厳選した車窓動画をユーザーに見せる。

### ■メインのターゲットユーザー

- 車窓からの景色を元々楽しんでいる人。
- 電車で旅行するのが好きな人。

### ■実装予定の機能（以下の例は実際のアプリの機能から一部省略しています）

1. 前面展望動画の再生機能 
    - ユーザーがアプリ内で電車の前面展望動画を再生できる。
2. 動画検索機能
    - ユーザーは路線名、都道府県を条件に検索することができる。
3. ユーザーはコメント機能
    - ユーザーが車窓動画に対してコメントをすることができる。
4. ログイン機能
    - ユーザーはログイン、ログアウトすることができる。
5. お気に入り登録
    - ユーザーは車窓動画をお気に入りに登録することができる。
6. マイページ機能
    - 登録しているユーザー情報（名前とメールアドレスのみ）を編集、更新、削除することができる。
    - お気に入り登録した動画を一覧表示することができる。

### ■なぜこのサービスを作りたいのか？

仕事中に見ていた石切〜額田間の眼下に広がる大阪平野の景色が本当に綺麗でお気に入りのスポットでした。しかし車内を見るとスマホに集中してしたり、景色に背を向けて座っていたりで気づいていない人がいる。またその景色が地元民の間だけでしか知られていないという現実を知りました。
そのようなことから、Youtube上にある”前面展望動画”を活用し、実際に電車に乗って景色を眺めているかのような体験をしていただくことで、電車から見える景色の良さを多くの人に知ってもらえると考えました。
